# Day016

## Main objective

Complete validation of the QM/MM electrostatic embedding workflow and generate production-quality diagonal excitonic Hamiltonians from embedded TDDFT calculations.

---

## Major developments

### ORCA electrostatic embedding

Implemented a robust electrostatic embedding workflow using external point charges extracted from the classical MD environment.

A critical debugging stage identified that ORCA expects the point-charge file in the format

```
N
q x y z
```

rather than

```
N
x y z q
```

After correcting the format, all embedded TDDFT calculations converged normally.

---

### Embedded TDDFT validation

Successfully completed embedded calculations for

- PYR2
- PYR3
- PYR4
- PYR5

including

- excitation energies
- oscillator strengths
- dipole moments

for multiple MD frames.

No geometry problems or short-contact artifacts were detected.

---

### Production pilot

Successfully processed

- frame000
- frame010
- frame020

for all four chromophores.

The production pipeline completed without failures.

---

### Site-energy trajectory

Generated

- raw site energies
- frame-centered energies
- globally centered energies

allowing direct analysis of environmental fluctuations along the MD trajectory.

Observed fluctuations remain within the expected meV range.

---

### Diagonal excitonic Hamiltonians

Generated

Hdiag(frame000)

Hdiag(frame010)

Hdiag(frame020)

together with metadata and summary tables.

These matrices constitute the diagonal contribution to the excitonic Hamiltonian

H(t)=Hdiag(t)+J(t)

that will be used in the next quantum-dynamics stage.
