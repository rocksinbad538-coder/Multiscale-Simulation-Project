# Day 017 — Scalable embedded TDDFT production workflow

## Objective

Promote the validated Day016 electrostatic-embedding TDDFT workflow from a controlled pilot to a scalable production workflow for MD-derived site-energy extraction.

## Starting point

At the start of Day017, the project had:

- 21 extracted MD frames from the accepted 100 ps trajectory.
- 12 completed embedded TDDFT calculations from Day016:
  - frames 000, 010, 020
  - chromophores PYR2, PYR3, PYR4, PYR5
- Validated ORCA point-charge embedding format.
- Validated extraction of S1 site energies.
- Existing diagonal Hamiltonians for frames 000, 010, and 020.

The embedding input generator was reused to expand the ORCA input set from the validated pilot to the complete production set.

Final input count:

- 84 ORCA input files.
- 84 point-charge embedding files.
- 84 QM-only pyrene XYZ files.

### 3. Resumable ORCA batch runner

A safe, resumable ORCA batch runner was implemented:

- `scripts/phase1A/run_day017_orca_embedding_batch.py`

The runner:

- detects frame and chromophore labels automatically;
- checks whether an output has already terminated normally;
- skips completed calculations;
- launches only missing jobs;
- reports return code, completion status, and runtime;
- avoids overwriting validated outputs unnecessarily.

### 4. Controlled production batch

A controlled production batch was launched for frames 001–003.

Executed jobs:

- frame001: PYR2, PYR3, PYR4, PYR5
- frame002: PYR2, PYR3, PYR4, PYR5

Result:

- 12/12 calculations completed.
- 12/12 normal ORCA termination.
- 12/12 successful TDDFT completion.
- No SCF failures.
- No embedding format failures.
- No manual intervention during the batch.

Total completed embedded TDDFT calculations after Day017 batch:

## Scientific interpretation

The production subset reproduces the pattern observed in the initial pilot:

- PYR2, PYR3, and PYR4 remain grouped near 4.00–4.02 eV.
- PYR5 is consistently red-shifted near 3.75–3.79 eV.
- The observed PYR5 detuning remains large relative to the short-time fluctuations in PYR2–PYR4.
- This validates the need to propagate the excitonic model with time-dependent diagonal disorder derived from embedded TDDFT rather than assuming static, uniform site energies.
