## Haken-Strobl audit

The Haken-Strobl implementation successfully preserved:

- trace
- Hermiticity
- population conservation
- positivity

The model correctly approaches a uniform long-time population distribution, consistent with pure dephasing.

The uniform asymptotic state is interpreted as a model artifact rather than physical thermalization.

## Lindblad exciton relaxation

An exciton-basis Lindblad model with detailed-balance structure was implemented and validated.

Expected thermal site populations at 300 K:

- PYR2 = 0.305432
- PYR3 = 0.160919
- PYR4 = 0.283158
- PYR5 = 0.250492

Numerical trajectories converge to these values with errors approximately 10^-9 for k_down ≥ 1 ps^-1.

## Initial-condition robustness

Initial excitation on:

- PYR2
- PYR3
- PYR4
- PYR5

All simulations converged to the same thermal distribution.

This confirms that the steady state is controlled by the Hamiltonian and bath model rather than the initial condition.

## J23 sensitivity

The weak hydrated value:

J23 = 2.37 meV

remains the primary bottleneck controlling transport.
