# Day 013 — Phase 1A Excitonic Hamiltonian and Hydration Effects

## Objective

Advance Phase 1A from electronic-structure inputs toward a first explicit excitonic-network model for the four-pyrene h-BN/water system.

## Completed work

### 1. ORCA TDDFT monomer calculations

Completed TDDFT calculations for isolated pyrene chromophores:

| Site | S1 energy (eV) |
|---|---:|
| PYR2 | 3.779 |
| PYR3 | 3.774 |
| PYR4 | 3.782 |
| PYR5 | 3.767 |

Vacuum site-energy disorder:

- 15 meV

### 2. ORCA TDDFT dimer calculations

Completed TDDFT calculations for all six pyrene dimers.

Preliminary effective couplings were estimated from the lowest dimer splitting:

J_eff = (E2 - E1) / 2

| Pair | J_eff (meV) |
|---|---:|
| PYR2-PYR3 | 6.5 |
| PYR2-PYR4 | 2.5 |
| PYR2-PYR5 | 8.5 |
| PYR3-PYR4 | 10.5 |
| PYR3-PYR5 | 1.0 |
| PYR4-PYR5 | 11.0 |

### 3. Geometry–coupling interpretation

The strongest couplings are not the long-range parallel pairs. Instead, the stronger couplings occur mostly among closer chromophore pairs.

This indicates that the excitonic network is controlled by full 3D chromophore geometry rather than by plane-angle classification alone.

### 8. Vacuum vs hydrated comparison

| State | Vacuum PR | Hydrated PR | Change |
|---|---:|---:|---:|
| X1 | 2.658 | 1.913 | -0.745 |
| X2 | 1.909 | 2.468 | 0.560 |
| X3 | 1.776 | 2.313 | 0.537 |
| X4 | 3.255 | 1.903 | -1.352 |

Interpretation:

Explicit local hydration red-shifts all pyrene S1 energies and increases site-energy disorder from 15 meV to 33 meV. Hydration redistributes the excitonic states and partially localizes some states, but does not collapse the network into fully localized single-site excitations.

## Main scientific conclusion

Day013 produced the first complete multiscale chain:

MD structure → TDDFT monomers → TDDFT dimers → excitonic Hamiltonian → diagonalized exciton states → hydration-induced disorder analysis.

The Phase 1A system behaves as a weakly coupled excitonic network with partially delocalized states. Local water modifies the site-energy landscape but does not eliminate excitonic mixing.

## Limitations

The current off-diagonal couplings are first-pass splitting-derived estimates. They should be refined using transition-dipole, transition-density, or fragment-based coupling analysis.
