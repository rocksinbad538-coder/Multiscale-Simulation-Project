## Geometry Inspection

Created `scripts/analysis/inspect_xyz_geometry.py` to verify the dimensions of Phase 0 generated geometries.

The script reads an XYZ file, estimates the lumen diameter, outer diameter, length, radial bounds, and coordinate bounds, and produces a cross-section figure for documentation.

Generated outputs:

- `results/geometry/generic_tube_24OD_14ID_20L_summary.csv`
- `figures/geometry/generic_tube_24OD_14ID_20L_cross_section.png`

This verifies that the initial generic tubular scaffold has the intended dimensions: 24 nm outer diameter, 14 nm lumen diameter, and 20 nm segment length.

## Candidate Matrix Documentation

Created `docs/phase0_candidate_matrix.md` to define the first Phase 0 candidate comparison.

The initial candidate classes are:

1. Organic / biomimetic tubular scaffold
2. Hybrid organic–inorganic tubular scaffold
3. Predominantly inorganic tubular scaffold

All candidates will initially use the same reference geometry: 24 nm outer diameter, 14 nm lumen diameter, 5 nm wall thickness, and 20 nm segment length.

The first inorganic baseline recommendation is BNNT with confined water because it provides a robust tubular scaffold with local B–N polarity and a more interpretable dielectric/confined-water environment than a conductive CNT baseline.


## Inorganic Baseline Modeling Decision

Documented the first major modeling decision for Phase 0: the predominantly inorganic baseline will initially be represented as a BNNT / BN-like tubular scaffold with confined water.

A key geometry issue was identified. A single-wall BNNT cannot directly reproduce the common Phase 0 geometry of 24 nm outer diameter, 14 nm lumen diameter, and 5 nm wall thickness because a single BNNT wall is effectively monolayer-like. Therefore, the first inorganic baseline will be a geometry-equivalent BN-like scaffold for controlled screening, with later refinement to explicit single-wall or multi-wall BNNT models if the inorganic baseline shows strong measurable response.
