# Day 005 — Matched carbon-like tubular scaffold branch

Day 005 starts the next Phase 0 architecture-screening branch.

The validated BN-like scaffold-water system is used as the reference state. A new matched carbon-like neutral tubular scaffold is generated using the same scaffold geometry and confined-water configuration, but replacing the alternating charged B/N-like scaffold atoms with neutral carbon-like scaffold atoms.

Purpose:

- isolate architecture/geometric confinement from wall polarity/charge-pattern effects;
- compare BN-like polar scaffold vs carbon-like neutral scaffold under identical geometry, hydration, temperature, and field/off-field protocols;
- prepare a direct Phase 0 architecture-screening comparison.

Initial Day 005 target:

- generate matched carbon-like scaffold-water data file from the validated BN-like contained state;
- run contained 300 K scaffold-water control;
- check stability, confinement, water radial distribution, and dipole orientation.

## Initial LAMMPS input correction

The first carbon-like scaffold-water run stopped during `read_data` with:

- `ERROR: Incorrect args for pair coefficients`

This was traced to coefficient sections inherited from the converted BN-like `.data` file. Since the Day 005 input explicitly defines the intended carbon-like scaffold-water coefficients, the input was corrected to use `read_data ... nocoeff`, allowing LAMMPS to read topology/coordinates/velocities while assigning coefficients directly from the input script.

## Day 005 field-response interpretation

The fieldZ comparison shows that all three water-containing systems respond strongly to the applied axial electric field. BN-like scaffold-water, carbon-like scaffold-water, and water-only controls all develop a similar axial dipole response, with increasing axial dipolar order and positive mean cos(theta_z).

The main architecture-dependent difference is radial confinement. Under fieldZ, the BN-like polar scaffold maintains the maximum radial water oxygen position near approximately 73 Å, whereas the carbon-like neutral scaffold reaches a much larger radial excursion, above 100 Å over the 1.0 ps field diagnostic. The water-only control also expands radially, but less than the carbon-like scaffold in this diagnostic.

This supports the Phase 0 interpretation that the BN-like wall polarity/charge pattern contributes to stronger radial confinement than geometry alone. The carbon-like neutral scaffold provides a useful matched-geometry control, but it does not reproduce the confinement behavior of the BN-like polar scaffold under either field-free or field-on conditions.

## BN-neutralized matched scaffold branch

To isolate the effect of scaffold charge/polarity from scaffold geometry and B/N-like topology, a BN-neutralized matched scaffold-water branch was prepared.

This branch preserves:

- BN-like scaffold geometry;
- scaffold atom types 1 and 2;
- water configuration;
- box dimensions;
- bonded water topology;
- simulation protocol.

The only intended scaffold modification is setting the B/N-like scaffold charges to zero. This provides a more direct charge-pattern control than the carbon-like neutral scaffold, which changes both charge and scaffold chemistry.

## BN-neutralized 20k matched-time extension

The BN-neutralized scaffold-water branch was extended to 20,000 total steps for matched-time comparison against the BN-like polar scaffold, carbon-like neutral scaffold, and water-only controls.

The extension completed successfully:

- total trajectory length: 20,000 steps
- timestep: 0.10 fs
- total simulated time: 2.0 ps
- dangerous builds: 0
- NaNs/lost atoms/missing bonds: none detected

At 20,000 steps, the BN-neutralized scaffold-water system reached a maximum radial water oxygen position of approximately 95.25 Å. This is much larger than the BN-like polar scaffold-water reference, which remains near approximately 73 Å, and is close to the carbon-like neutral scaffold behavior.

This indicates that neutralizing the BN-like scaffold charges substantially weakens radial water confinement while preserving the same scaffold geometry and B/N atom-type topology. Therefore, the strong radial confinement observed in the BN-like polar scaffold is not explained by geometry alone; the scaffold charge/polarity pattern is a key Phase 0 design variable.

This indicates that the BN-like charge/polarity pattern is a key design variable for radial confinement, while the applied field controls the average axial dipole alignment.
