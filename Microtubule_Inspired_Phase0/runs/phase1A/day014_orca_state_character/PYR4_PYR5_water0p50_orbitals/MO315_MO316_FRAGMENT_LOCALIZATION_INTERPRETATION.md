# MO315/MO316 Fragment Localization Interpretation

## System

PYR4-PYR5 hydrated dimer with local 0.50 nm water shell.

## Relevant TDDFT state

The low-energy S1 state at 2.959 eV is dominated by:

- MO315a -> MO316a
- excitation weight = 0.999988
- oscillator strength = 0.000000

## Fragment localization

| MO | PYR4 fraction | PYR5 fraction | Water fraction |

## Interpretation

MO315a is localized almost entirely on PYR4, while MO316a is localized almost entirely on PYR5. Therefore, the anomalous S1 state is best interpreted as an inter-chromophore PYR4 -> PYR5 charge-transfer-like state, not as a water-localized artifact.

Because this state is dark and charge-transfer-like, it should not be used directly as a simple Frenkel excitonic splitting estimate.

For the hydrated PYR4-PYR5 Frenkel-like coupling estimate, the more appropriate first-pass bright monomer-like pair is:

- S2 = 3.746 eV
- S5 = 3.812 eV
- splitting = 0.066 eV
- J_eff ≈ 33 meV

This is larger than the previous vacuum first-pass PYR4-PYR5 estimate of approximately 11 meV.
