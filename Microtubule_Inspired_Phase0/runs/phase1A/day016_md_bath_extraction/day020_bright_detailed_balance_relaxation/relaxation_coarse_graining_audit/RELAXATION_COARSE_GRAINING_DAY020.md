# Day020 Relaxation Coarse-Graining Audit

## Purpose

This audit tests whether the rapidly mixing PYR2-PYR4 manifold can be represented as a single upper reservoir coupled to the low-energy PYR5 sink.

## Validation

- Hamiltonians: 21/21.
- Minimum assigned site weight: 0.57938732.
- Maximum generator column-sum error: 6.661e-16.
- Maximum detailed-balance error: 4.441e-16.

## Coarse-graining result

- Minimum internal-mixing/capture rate ratio: 7.072e+01.
- Maximum relative error between the coarse-grained capture rate and the exact slow mode: 9.088e-03.
- PYR4 gateway contribution range: 0.831842 to 0.875653.

## Interpretation

A large separation ratio supports rapid pre-equilibration within PYR2-PYR4 followed by slow leakage toward PYR5. Agreement with the exact slow mode validates the reduced upper-reservoir plus PYR5-sink description.

All absolute capture times remain proportional to the phenomenological kappa parameter and are not microscopic predictions.
