# Day012 ORCA TDDFT Pilot Summary

This directory contains compact summaries from the first ORCA TDDFT pilot campaign.

The calculations use PBE0/def2-SVP with TightSCF and SlowConv.

The campaign includes:

- four isolated pyrene monomers
- two primary pyrene dimers

The main scientific result is the observation of low-energy dimer excited states below the monomer S1 energies, consistent with preliminary excitonic-state splitting.
