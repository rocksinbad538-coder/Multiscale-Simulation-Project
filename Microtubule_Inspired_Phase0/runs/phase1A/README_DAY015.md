# Phase 1A — Day015 Outputs

## Purpose

Day015 contains open-system exciton dynamics, Lindblad thermalization, bath-parameter mapping, and synthetic dynamic-disorder scans for the hydrated PYR2-PYR5 Hamiltonian.

## Main folders

- `day015_exciton_model/`: four-site exciton Hamiltonian and Haken-Strobl results.
- `day015_lindblad_relaxation/`: Lindblad relaxation dynamics and thermal-equilibrium audits.
- `day015_lindblad_initial_conditions/`: initial-condition robustness scans.
- `day015_J23_sensitivity/`: PYR2-PYR3 coupling sensitivity.
- `day015_PYR2_detuning_sensitivity/`: PYR2 energetic detuning sensitivity.
- `day015_bath_parameter_mapping/`: MD-to-bath proxy workflow and input specifications.
- `day015_dynamic_disorder_lindblad/`: synthetic dynamic-disorder ensemble.
- `day015_dynamic_disorder_sigma_tau_map/`: sigma_E versus tau_c map.
- `day015_dynamic_disorder_gamma_sigma_map/`: gamma_phi versus sigma_E map.
- `day015_open_system_comparison/`: consolidated open-system comparison.

## Key conclusion

Synthetic dynamic site-energy disorder identifies an ENAQT-like transport-enhancement regime. The next step is MD-derived bath calibration.
