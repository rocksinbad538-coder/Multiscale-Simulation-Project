# Day015 J23 Sensitivity Analysis

## Purpose

Test whether the weak PYR2-PYR3 coupling is the dominant bottleneck controlling access to PYR2 in the hydrated four-site Hamiltonian.

## Fixed parameters

- J34 = 17 meV
- J45 = 33 meV
- gamma = 10 ps^-1
- initial condition: PYR5 population = 1

## Interpretation

If PYR2 access is bottlenecked by J23, then maximum PYR2 population should increase and first-arrival times should decrease as J23 increases.
